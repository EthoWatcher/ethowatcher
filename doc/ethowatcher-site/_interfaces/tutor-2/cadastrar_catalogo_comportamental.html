Descrição

O módulo de cadastro de catálogos serve para que o usuário possa registrar catálogos comportamentais categóricos, esses podem ser utilizados posteriormente na transcrição comportamental categórica. Esse módulo permite ainda, cadastrar para cada comportamento um nome, uma descrição e um atalho alfanumérico. A interface gráfica pode ser vista na figura 55 (a função de cada componente na interface gráfica da figura 55 está apresentado a seguir).

  • Campo “name” – Nesse campo o usuário deve informar o nome da categoria comportamental e pode conter qualquer alfanumérico e qualquer quantidade de caracteres. Esse campo é obrigatório ser preenchido.
  • Campo “code” – Nesse campo, o usuário deve inserir o código correspondente ao comportamento que será criado. Para esse código ser usado como atalho do teclado ele deve conter no máximo um carácter alfabético. O preenchimento desse campo também é obrigatório para a criação de um novo elemento do catálogo.
  • Botão “Description” – Nesse campo, o usuário deve inserir a descrição do comportamento que será criado. O preenchimento desse campo também é obrigatório para a criação de um novo elemento do catálogo. O campo pode conter qualquer alfanumérico e qualquer quantidade de caracteres.
  • Tabela “behavior catalog units” – Nessa tabela, são apresentadas as categorias criadas para o novo catálogo. Cada linha corresponde a uma categoria, sendo a primeira coluna referente ao nome dessa categoria e a segunda referente ao seu código.
  • Botão “add to catalog” – Ao clicar nesse botão, o usuário cria um novo comportamento para o catálogo. A criação do elemento somente será finalizada se os campos “Name” e “Code” estiverem preenchidos.
  • Botão “delete selected item” – Ao clicar nesse botão é apagada a categoria selecionado na tabela. Logo, o elemento correspondente a esse item é excluído do catálogo. Para selecionar um item, deve-se clicar sobre uma das linhas contendo o comportamento desejado.
  • Botão “save/ register catalog”– ao clicar nesse botão é aberto uma caixa de diálogo com acesso ao disco local do computador para que o usuário possa escolher onde será criado o novo arquivo de catálogo comportamental
