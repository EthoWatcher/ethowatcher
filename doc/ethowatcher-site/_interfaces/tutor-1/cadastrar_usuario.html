Descrição

O módulo de cadastro de usuário permite ao usuário cadastrar seu nome, seu laboratório e inserir uma senha. As informações sobre o nome do experimentador e seu laboratório, são gravadas em todos os arquivos XML que o EthoWatcher OS gera: arquivos de cadastro de vídeo, arquivos de catálogo de comportamentos, análise etográfica dos vídeos e nas análises de concordâncias. A senha junto com a informação de nome e do laboratório, são usadas para gerar a tranca no módulo de embaralhamento e habilitar a revelação das informações cegadas no módulo de retirada de véu. A interface gráfica pode ser vista na figura 54 (a função de cada componente na interface gráfica da figura 54 está apresentada a seguir).

Campo “User ID” – Nesse campo o usuário deve informar o seu nome, e pode conter qualquer alfanumérico de qualquer tamanho. Esse campo é obrigatório ser preenchido. • Campo “Lab Info” – Nesse campo o usuário deve informar o nome de seu laboratório e pode conter qualquer alfanumérico de qualquer tamanho. Esse campo é obrigatório ser preenchido. • Campo “Password” – Nesse campo o usuário deve informar uma palavra-chave e pode conter qualquer alfanumérico de qualquer tamanho. Esse campo é obrigatório ser preenchido. • Botão “Save” – ao clicar nesse botão é aberto uma caixa de diálogo com acesso ao disco local do computador para que o usuário possa escolher onde será criado o novo arquivo de usuário.
