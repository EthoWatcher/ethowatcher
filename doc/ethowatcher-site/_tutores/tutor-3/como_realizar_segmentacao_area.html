Como realizar o rastreamento do vídeo

Nesse tutorial iremos descrever passo a passo como utilizar o modulo de rastreamento. Para fazer esse tutorial é importante que você já tenha feito o tutorial de como realizar uma etografia. Como no tutorial passado nesse você ira precisar ter baixado do o vídeo do relógio.

Como cadastrar um vídeo e configurar para realizar o rastreamento

Nesse tutorial será focado em como configurar, calibar o rastreamento do vídeo da forma mais simples possível. Como o EthoWatcher OS ainda é possivel configurar formas de tentar suprimir o ruido, e de fazer o processamento de diferentes areas de interese. Como configurar tais algorítimos será discutido em outros posts.
