Descrição

  • Botão de “register new user” – ao clicar nesse botão o programa abre uma janela para cadastrar um novo usuário. A interface que é aberta é apresentada no tópico 1.1
  • Botão de “load user” – ao clicar nesse botão o programa abre uma caixa de diálogo com acesso ao disco local do computador. Isso acontece para que o usuário possa encontrar um arquivo de usuário (gerado pela interface 1.1) esse arquivo irá de cabeçalho nas transcrições, análises, e segmentações.
  • Botão de “Register Behavioral Catalog” – ao clicar nesse botão o programa abre uma janela para cadastrar um novo catálogo comportamental. A interface aberta é apresentada no tópico 1.2.
  • Botão de “Register Video files” – o clicar nesse botão o programa abre uma janela para cadastrar um novo vídeo que Figura 51: Interface do Ethowatcher OS – Aba das ferramentas de cadastro e gestão de vídeo.possa ser utilizado no Ethowatcher Open-Source. A interface aberta é apresentada no tópico 1.2
  • Botão de “Build video file list” – ao clicar nesse botão o software abre uma nova janela para cadastro de uma lista de arquivos de vídeos. A interface aberta é apresentada no tópico 2.1.
  • Botão de “Build video file list” – ao clicar nesse botão o software abre uma nova janela para cadastro de uma lista de arquivos de vídeos. A interface aberta é apresentada no tópico 2.1.
  • Botão de “Blind video files” – ao clicar nesse botão o software abre uma nova janela de embaralhamento de vídeo. A interface aberta é apresentada no tópico 2.2.
  • Botão de “Unveil blinded video files” – ao clicar nesse botão o software abre uma nova janela para o desembaralhamento dos vídeo cegos. Essa interface permite retirar o véu posto nos arquivos de vídeo. Essa interface é apresentada no tópico 2.3.
  • Botão de “build behavioral-sample video” – ao clicar nesse botão o software abre uma janela para que o usuário possa segmentar o relatório de TCC e TCCM por comportamento. Essa interface é apresentada no tópico 4.4.
  • Botão “EthoWatcher” na aba superior – ao clicar nesse botão o usuário é direcionado para uma tela que mostra a lista de desenvolvedores do EthoWatcher e sua licença de uso.
