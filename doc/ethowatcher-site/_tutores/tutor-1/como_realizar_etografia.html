Como realizar uma etografia

Em um software etografico uma das primeiras funcionalidades a ser explorada no EthoWatcher OS é a de realização de uma etografia. Nesse tutorial iremos demonstrar como é possivel cadastrar um catalogo comportamental, registrar um vídeo de comportamento animal e realizar uma etografia. Para acompanhar esse tutorial voce já deve ter baixado o programa e o vídeo sobre o relógio. figura 1. Caso você ainda não tenha baixado algum desses componentes vá para a pagina de download

O vídeo exemplo, figura 1, é composto por um relógio preto posicionado em um fundo preto. Esse relógio possue uma espuma retangular branca acoplada no seu ponteiro de tal forma que a maior largura dessee retagunlo esteja parela a orientação do ponteiro. Isso é feito para possuir um grande contraste entre o objeto de interesse e o fundo. O relógio está cofinado dentro de uma arena de nado forçado. O vídeo possue uma visão de cima para baixo e foi gravado 30 amostras por segundo. Nesse tutorial iremos analisar o comportamento desse relógio. Para começar nossa análise comportamentla do relógio o primeiro passo é definir nosso catalogo comportamental. Para isso abra o EthoWatcher OS, figura 2, e clique no botão "Register Behavioral Catalog" para abrir o módulo de cadastro de catalogo comportamental Ao clicar nesse botão será aberto a interface gráfica de cadastro de catalogo comportamental.

Passo 1- Cadastrar uma catalogo comportamental

Após observar o comportamento do relógio pelo Media Player, vem junto com o pacote k-lite-code pack, determinaos que o relógio possue 4 grandes categorias comportamentais. Essas categorias dependem da orientação do ponteiro do relógio. Para melhor descrever essas categorias imaginamos que o relógio em cena possue as suas numerações, assim determinamos que :

  • a categoria denominada "Deslocamento123" acontece quando o ponteiro do relógio está se deslocando entre os numero 12 e o 3.
  • a categoria denominada "Deslocamento36" acontece quando o ponteiro do relógio está se deslocando entre os numero 3 e o 6.
  • a categoria denominada "Deslocamento69" acontece quando o ponteiro do relógio está se deslocando entre os numero 6 e o 9.
  • a categoria denominada "Deslocamento912" acontece quando o ponteiro do relógio está se deslocando entre os numero 9 e o 12.

Cadastrando uma categoria

Para cadastrar essas categorias comportamentais vá para a interface gráfica de cadastro de catalogos, figura 3, e repita para cada categoria comportamental de :

  • preencher o nome da categoria no campo "Name" (e.g. Deslocamento36)
  • Adicionar uma tecla única do teclado no campo "shortcut" para ser um atalho (e.g. "d"). Esse shortcut não se deve repetir entre as categorias e ele só pode possuir uma tecla
  • Adiconar a descrição desse comportamento (e.g., "deslocando entre os numero 3 e o 6.")
  • Apertar no botão "add new categoria"

Deletando uma categoria

Caso você tenha adicionado alguma categoria por engano é possivel deletar as categorias do catalogo. Para isso

  • Selecione a categoria na tabela "Behavioral catalog units"
  • Clique no botão delete selected item

    • Salvando o catalogo comportamnel

    Após concluir o cadastro de todas as categorias clique no botão "Save/Register catalog" para salvar o catalogo comportamental. Irá abrir uma janela que permitirá nomear o catalogo comportamental construido. Sugerimos que você grave isso dentro da pasta do caminho "C:/EthoWatcher/catalogo".

    Passo 2- Cadastrar um vídeo de comportamento animal para a etografia.

    Após realizarmos a construção do nosso catalogo comportamental é necessário cadastrar algumas informações importantes a respeito do vídeo experimental. Com o EthoWatcher OS é possivel registrar o vídeo comportamental para ser usado no modulo de tracking, modulo de cegamento e para realizar etografias. Você pode conferir as outras formas nos outros tutoriais. Nesse tutorial, como é o primeiro, iremos cadastrar o vídeo apenas para realizar a etografia. Abra a interface de cadastro de vídeo, para isso vá para a tela inicial do EthoWatcher OS, figura 4, e clique no botão register vídeo files.

    Abrindo vídeo

    Para abrir o vídeo clique no botão "load vídeo". Isso fará com que abra uma interface gráfica para você navegar pelas pastas do seus computador. Encontre o vídeo de interesse. Caso você tenha seguido as nossas sujestões esse vídeo estara em "c:/ethowatcher/videos/". Após encontrar clique em open.

    IMPORTANTE

    É importante que o caminho e o nome do vídeo possuam apenas caracters da tabela ASCII. Isso significa que não pode ter caracteres especiais como "éãç"

    Caso os codificadores do vídeo estejam corretamente instalado o vídeo será aberto e os campos do "Step 1.02 Video Open Info" estarão preenchidos corretamente. O campo :

    • Video file path, informa o caminho que se encontrar o vídeo no sistema operacional
    • Video file name, informa o nome do vídeo
    • frames per second (FPS), informa a frequencia de amostragem que o filme possue.
    • widht (px), informa a lagura em pixeis dos quadros que o vídeo possue
    • height (px), informa a altura em quantidade de pixeis dos quados que o vídeo possue

    Configurando o regristro de vídeo

    Como mencionado anteriormente é possivel registrar o víodeo para ser usado em uma série de módulos. Para cada módulo é necessário fazer uma série a mais de passos. Nesse tutorial registraremos o vídeo apenas para análise etografica. Para isso deixe desmarcado as opções:
    • tracking and Morphological/kinematica analysis, permite cadastrar o vídeo para realizar o rastreamento
    • Blinding video files, permite realizar o cegamento do vídeo.
    Clique no botão "Next Step."

    Selecione os quadros importantes

    Toda análise etografica possuí um momento que se inicia e um momento que ela é finalisada e muitas vezes, inclusive é recomendado, que esses quadros não sejam o primeiro ou o ultimo. Para isso é importante cadastrar esses quadros. No nosso exemplo iremos realizar o registro de dois minutos (120 segundos) de comportamento do relógio. Ela começara em 2430 (81.081) e terminará no quadro em 6026 (201.068 seconds). Para cadastrar esses quadros utilizando o EthoWatcher OS você ira precisar navegar pelo vídeo. Para isso o EthoWatcher possui 3 formas de navegação pelos quadros do vídeo:

    • navigation, esse é a navegação por passos, você pode retroceder e avancar uma quantidade fixa de quadros. Esses quadros podem ser ajustados
    • controls, essa navegação é frame a frame e é continua, a velocidade pode ser alterada
    • scrol bar, voce pode navegar clicando e segundo o scroll do vídeo
    Para o nosse exemplo, então, utilizando os navegadores do vídeo vá até o quadro 2430 e clique no botão "Frame to Start the Analyses" para cadastrar esse quadro como o primeiro quadro da análise. Após navegue pelo vídeo e vá para o quadro 6026 e clique no botão "Frame to finish the anlysis". Após, cadastrar os quadros importantes para análise clique em "Next Step".

    Identifique a sessão

    Após cadastrar os quadros importante o EthoWatcher OS permite, como passo adicional, adicionar informações a respseito do experimento. No campo:

    • Experiment, você pode descrever o experimento
    • Date, a data que foi realizado o experimento
    • Other info, outras informações relevantes ao experimento
    • Animal ID, o id do animal
    • Weight, o peso do animal
    • Sex, osexo do animal
    • Treatment/condition, o tratamento e a condição do animal

    No nosso exemplo, preencha o campo "Experiment" com o valor clock, coloque no campo Date o dia 03/05/2019, escreva "None" em "Other Infos". Também, preencha os dados de nosso objeto mecanico. Coloque no campo "Animal ID" o valor 1, o weight o valor de 10, e no campo "sex" o valor de male, e por final no campo Trheatment Condition coloque o valor de control.

    Após realizar esses passos clique no botão "Save"

    Uma vez que o vídeo for cadastrado evite de ficar modificando ele de caminho.

    Passo 3- Realizar a etografia.

    Agora que já cadastramos o vídeo e possuímos o catalogo o próximo passo é realizar etografia do vídeo. Para abrir a janela de etografia baseada em vídeo vá para a tela inicial do EthoWatcher OS, clique na aba Ethographic and tracking tools e clique no botão video-based etography. Com isso, irá abrir a janela de Etografia baseada em vídeo, figura 7.

    Configurando a etografia baseada em vídeo

    Com a janela de etografia basea em vídeo aberto clique no botão "Open video". Esse botão irá abrir uma janela que permite localizar os vídeos etograficos que foram registrados. Encontr o registro de vídeo que foi realizado no passo anterior. Em seguida, vai aparecer 3 botões: video-based ethography, Trackin, Video based ethography and tracking; porém somento o botão "video-based ethography" estará liberado para ser clicado. Isso acontece porque no registro de vídeo sómente configuramos para realizar etografias. Clique no botão habilitado. Ao clicar no botão "Video based ethography" irá aparecer para abrir um catalogo comportamental. Clique no botão "Open Behavioral catalog" para encontrar o catalogo comportamental previamente cadastrado. Esse botão também abre uma janela gráfica que permite se locomover pelas pastas do computador. Encontre o catalago cadastrado no passo 1. Após selecionar o catalogo só falta informar se a sintax do record comportamental possuí evento auto-exclusivos, ou eventos sobrepostos. Os eventos auto-exclusivos só permitira voce registrar uma categoria por vez. Já os eventos sobrepostos permitem registrar mais de uma categoria por vez. No nosso exemplo selecione "auto-exclusive events". E eclique no botão "Start."

    Limitação : Todas as análises posteriores da etografia envolvem que a sintax seja de eventos auto-exclusivos

    Realizando a etografia

    Finalmente, chegamos na janela para realizar a etografia. Como no registro de vídeo é possivel realizar a etografia utilizando os navegadores de vídeo. Para registrar uma categoria basta clicar no botão descriminado pelo seu nome na aba "etography" e essa categoria marcara todos os quadros no qual o botão estiver ativado. Também, é possivel selecionar a categoria a ser marcada usando os atalhos registrados no catalogo. Esses atalhos também são informados nos botões das categorias entre parenteses. Como nosso análise é auto-exclusiva só será permitido registrar uma cateogira por vez. A categoria que estiver ativada estará colorida com em vermelho. Realize a análise conforme o catalogo construído.

    Gravando a analise e gerando o relatório

    Após realizar a transcrição do vídeo em categorias vá para a aba Exit and Save e clique no botão "Save Video-based ethography" nesse botão te permitira selecionar uma pasta e um nome para essa análise etografica. Clique então em save. O EthoWatcher irá gravar dois arquivos em seu computador um com formatação XML, que será usado em futuruas análises e um em CSV que pode ser abrido por qualquer probrama de planilhas.

    O relatório é gravado no formato XML que pode ser usado nos módulos de análise. Ou o relatório pode ser fácilmente inspecionado e usado abrindo oarquivo CSV.

    Abrindo o relatório

    Para abrir o relatório recomendamos o uso do open office. A separação de celulas é por ";", figura 7; Dentro do relatório CSV é possivel ver o log de quando se iniciou cada categoria. e por quanto tempo ela durou. Também se encontra uma tabela descriminando a duração, frequencia e latencia de cada categoria. É possivel segmentar esse relatório por tempo.
