Como realizar uma etografia

Nesse tutorial iremos demonstrar como cadastrar um catalogo comportamental, registrar um vídeo de comportamento animal e realizar uma etografia. Para acompanhar esse tutorial voce já deve ter baixado o programa e o vídeo de amostra.Caso você ainda não tenha baixado um deses componetes vá para a pgina de download

Para começar o processo abra o EthoWatcher OS e clique no botão Register Behavioral Catalog. Após clicar nesse botão será aberto a interface de cadastro de catalogo, figura 1.

Passo 1- Cadastrar uma catalogo comportamental

É possivel realizar o cadastro de catalogo de categorias.

  • Para isso preencha o nome da categoria no campo Nome
  • Adicione uma tecla do teclado para ser um shortcut
  • Adicone a descrição desse comportamento
  • Aperte no botão add new categoria

Deletar uma categoria

É possivel deletar as categorias do catalogo. Para isso

  • Selecione a categoria na tablea x
  • Clique no botão

    • Salvando o catalogo comportamnel

    Realize a construção do catalogo, após isso clique no botão Sava/register catalog

    Passo 2- Cadastrar um vídeo de comportamento animal para a etografia.

    É possivel registrar o vídeo comportamental para ser usado no modulo de tracking, modulo de cegamento e para realizar etografia. Você pode conferir as outras formas nos outros tutoriais. Nesse tutorial, como é o primeiro, iremos cadastrar o vídeo apenas para realizar a etografia. Para isso

    • Cliqe no botão load video e confira
    • No passo 1.03 descelecioone as opções "Tracking and Morphological/kinematic analysis"
    • Blindig video file
    • Clique no botão next step

    Selecione os quadros importantes

    É importante selecionar quadros importantes para a análise

    Para navegar pelos quadros é possivel utilizar os controle. Naveue pelos quadros e:

    • Encontre o quadro de inicio da análise e clique no botão
    • Econtre o quadro final da análise e clique no botão Frame to finish the analysis
    • Ao encontrar esses quadros clique no botão Next Step

    Identifique a sessão

    Como passo complementar é possivel cadastar informações a respseito do experimento. No campo:

    • Experiment, você pode descrever o experimento
    • Date, a data que foi realizado o experimento
    • Other info, outras informações relevantes ao experimento
    • Animal ID, o id do animal
    • Weight, o peso do animal
    • Sex, sexo do animal
    • Treatment/condition, o tratamento e a condição do animal

    Após realizr esses passos cliqeu no botão Save

    Passo 3- Realizar a etografia.

    Para configurar a etografia que foi feita é necessário:

  • Clicar no botão open video, para informar o registro de vídeo
  • Informar que tipo de transcirção comportamental será feita

    • Realizando a etografia

    Para registrar a categoria voce pode utilizar os atalhos do teclado registrado ou clica no botão. Para navegar pelos quadros voce pode usar os controle é possivel diminuiir a velocidade de reprodução do vídeo ou ir manualmente avançando ou retrocedendo o v´deio.

    Passo 4- Gerar o relatório

    É poiissivel geral o relatório da etografia

    O relatório é gravado no formato XML que pode ser usado nos módulos de análise. Ou o relatório pode ser fácilmente inspecionado e usado abrindo oarquivo CSV.
